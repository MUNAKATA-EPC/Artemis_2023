## 全国機ロボット概要
* メインマイコン：Teensy 4.0
* サブマイコン：ATMega2560(ボールセンサ、ラインセンサ処理用)、Seeeduino Xiao(ジャイロセンサ処理用)
* モータードライバ：DSR-1202
* モーター：ジョンミンモーター x4
* IRセンサー：16個(円形)
* ラインセンサー：S4282-51 x25 + LED
* ジャイロセンサー：MPU6050(used dmp)
* カメラ：OpenMV H7
* 電源：Lipo 3cell(11.1V, 1300mAh)
